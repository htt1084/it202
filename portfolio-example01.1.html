Includes RSA Encryption/ Decryption, OOP, Java

Introduction

Create a Java program that uses RSA encryption/decryption to send messages from one user to another. The project will require a Graphic User Interface that will incorporate a number of tasks needed to perform the RSA Algorithm. These tasks/operations will include:

  • creating a public-key/private-key pair
  • “blocking” a large message into manageable sized chunks and its reverse
  • performing the encrypting/decryption operation.

User would have the options of:

  • encrypting a message
  • decrypting a message
  • blocking a message, with option to choose any block size (of a power of 2, i.e. 2, 4, 8, 16, 32, etc.)
  • unblock a message
  • create key pairs
    • from user given prime numbers
    • randomly chosen prime numbers
The details of project's description can be find here.

Initial Ideas

The main classes of the project are BigInt, RSA, PublicKey, and PrivateKey. The BigInt class handles the operations required to (add, subtract, multiply, etc.) the numbers resulting from encrypting a message. The RSA class is the main class of the project. It will contain all the files necessary to use the program (blockedMessageFile, PublicKeyFile, PrivateKeyFile) as well as the encrypt and decrypt functions. The PublicKey and PrivateKey classes (which we may combine into one), would be the class containing those functions that help us read the keys from their prospective files, as well as variables to hold them once read in. The last class, KeyPairGenerator, is a helper class (an extension of the PublicKey and PrivateKey classes) that contains the functions that compute the key pairs.
